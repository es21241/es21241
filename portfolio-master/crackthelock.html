GIT, GitHub, GitLab

GIT

CLion ir IDE, ko izmanto C un C++ valodu programmēšanai. Studiju laikā es to izmantoju laboratorijas darbiem un algoritmu analīzei. Tā sniedz spēcīgu atbalstu kompilācijai, debugošanai un koda analīzei.

GitHub

GitHub ir platforma Git repozitoriju glabāšanai un sadarbībai. Es publicēju savus projektus, veidoju dokumentāciju un izmantoju to komandas darba laikā ar pull requestiem un issue pārvaldību.

GitLab

GitLab ir līdzīga platforma kā GitHub, taču biežāk tiek lietota privātā vidē. Universitātē mēs to izmantojam kursu darbiem un uzdevumu nodošanai ar CI/CD atbalstu.
