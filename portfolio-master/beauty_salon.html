Virtual Box, Linux, Postman

Linux

Linux ir atvērtā koda operētājsistēma, ko bieži izmanto programmēšanā, serveru administrēšanā un drošības testos. Universitātē mēs izmantojam Linux, lai apgūtu komandrindas darbu, failu struktūru, lietotāju pārvaldību un skriptu veidošanu. Es strādāju gan ar Ubuntu, gan ar Kali Linux virtuālajās mašīnās, izmantojot tādus rīkus kā bash, vim, chmod, gcc, kā arī izmēģinot tīkla konfigurācijas un servera uzstādījumus. Linux man palīdz labāk izprast sistēmas līmeņa darbību un kļūst par svarīgu ikdienas darba daļu programmētāja ceļā. Linux ir Windows alternatīva.

Virtual Box

VirtualBox ļauj palaist virtuālās mašīnas. Es to izmantoju, lai testētu dažādas operētājsistēmas, mācītos Linux un veiktu drošības eksperimentus izolētā vidē. Piemēram kursā Object Oriented Programming mēs izmantojām VirtualBox, lai palaistu Ubuntu, jo tas ir daudz vieglāk nekā uzstādīt to uz sava datora.

Postman

Postman ir rīks API testēšanai. Programmēšanas kursos es to izmantoju, lai sūtītu HTTP pieprasījumus, analizētu atbildes un saprastu, kā strādā servera un klienta mijiedarbība. To izmantojām backend izstrādē priekš dažādam mājaslapam.
